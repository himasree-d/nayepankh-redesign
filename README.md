# NayePankh Foundation — Website Redesign

A single-page concept redesign for [NayePankh Foundation](https://nayepankh.com/), a UP Government registered NGO (80G & 12A certified) working across Kanpur, Ghaziabad, and other cities in India.

Built as part of the Front End Development internship task for NayePankh Foundation.

## Overview

The redesign reimagines the foundation's homepage as a single, cohesive page with a warm, editorial visual identity built around the "Pankh" (wing/feather) theme — small acts of help adding up to flight for someone in need.

## Sections

- **Hero** — Mission statement, key stats (lives impacted, cities reached, tax-exempt status), and call-to-action
- **About** — Foundation story and founder's quote
- **Programs** — Core initiatives: food drives, education support, sanitary pad donation, clothing drives
- **Impact** — Key numbers presented as a visual strip
- **Volunteer / Join** — Why and how to get involved
- **Contact & Donate** — Contact details, social links, and donation CTA

## Features

- Fully responsive (mobile, tablet, desktop)
- Dark mode toggle
- Scroll-triggered reveal animations
- Accessible focus states and reduced-motion support
- No frameworks — pure HTML, CSS, and vanilla JS

## Tech Stack

HTML5, CSS3, JavaScript (Intersection Observer for animations)

## Suggested Improvements for the Live Site

- Add a dedicated "Programs" page detailing each initiative with photos and outcomes
- Add an impact dashboard showing donation usage and program reach over time
- Add an online volunteer sign-up form instead of routing to the donate page
- Improve mobile navigation and add a sticky donate CTA
